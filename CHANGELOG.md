# История изменений

Формат основан на принципах [Keep a Changelog](https://keepachangelog.com/ru/);
версионирование — семантическое.

## [1.0.0] — 2026-06-11

Первая публичная версия (срез проекта к предзащите ВКР).

- Полный конвейер open-set re-ID тритонов по узору брюшка: кроп брюшка (YOLO seg + pose) →
  ориентация (голова/хвост) → распрямление ленты (`unroll_ribbon`) → эмбеддер
  **MegaDescriptor-L-384** (zero-shot) + косинусный поиск top-K → матчинг созвездия центроидов
  пятен → калиброванный %-score + решение «известная / новая особь» + оверлей совпавших пятен.
- **Streamlit-демо** (`app/demo.py`): опознание, top-K кандидатов, метка known/new, оверлей пятен.
- Финальная оценка на однократно вскрытом sealed-test (`artifacts/ab_test_headline.json`):
  overall recall@1 0.250 / @5 0.446; PW 0.818 / 1.000; TK temporal 0.188 / 0.386; open-set AUROC 0.446.
- Лицензия **AGPL-3.0-or-later**; публичная документация и UML-диаграммы — `docs_public/`.

### История разработки (этапы до 1.0.0)

- **0.1** — слой данных: манифест, md5-дедупликация, детерминированные сплиты
  (gallery/probe, train/dev/test), гейты валидации, CLI.
- **0.2** — сегментация и кроп брюшка + ориентация (YOLO seg/pose; разметка полуавтоматом
  SAM 2 / FastSAM с ручной правкой).
- **0.3** — распрямление узора в «ленту» (`unroll_ribbon`); A/B вариантов кропа.
- **0.4** — эмбеддер re-ID (MegaDescriptor-L-384; cross-fit OOF; open-set AUROC; дообучение
  оценено и отвергнуто — значимого прироста нет).
- **0.5** — матчинг созвездия центроидов пятен (guided); интерпретируемый оверлей.
- **0.6** — гибрид (эмбеддер + матчинг пятен) + калибровка %-score + решение known/new.
- **0.7** — Streamlit-демо, UML-диаграммы, раздел лицензий и источников.
