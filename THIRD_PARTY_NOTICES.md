# Сторонние компоненты и источники данных

Проект использует перечисленные ниже сторонние модели, библиотеки и датасеты.
Файл фиксирует их лицензии и условия атрибуции.

## Модели и библиотеки

| Компонент | Роль в проекте | Лицензия |
|---|---|---|
| **MegaDescriptor-L-384** (`BVRA/MegaDescriptor-L-384`, HF Hub) | эмбеддер re-ID (финальная система) | CC-BY-NC-4.0 |
| **MiewID** (`conservationxlabs/miewid-msv3`, HF Hub) | второй эмбеддер (только сравнительный эксперимент, не в финальной системе) | не объявлена (de facto all rights reserved) |
| **Ultralytics YOLO** (seg + pose) | сегментация брюшка + keypoints | AGPL-3.0 |
| **FastSAM** (Ultralytics) | бутстрап псевдо-масок (только этап разметки, offline) | AGPL-3.0 |
| **SAM 2** (Meta) | точечная пересегментация в разметчике (опционально) | Apache-2.0 |
| **timm** (PyTorch Image Models) | загрузка бэкбона | Apache-2.0 |
| **PyTorch** | DL-фреймворк | BSD-3-Clause |
| **FAISS** (Meta) | векторный поиск top-K — задел для масштабирования; в текущей версии не используется (поиск на numpy-косинусе) | MIT |
| **OpenCV** | CV-операции | Apache-2.0 |
| **scikit-image** | blob-детекторы (LoG/DoG) | BSD-3-Clause |
| **albumentations** | аугментация | MIT |
| **NumPy / pandas / PyYAML / pycocotools** | инфраструктура | BSD-3 / BSD-3 / MIT / BSD |

Примечания:

- Лицензия CC-BY-NC-4.0 на веса MegaDescriptor ограничивает **коммерческое** использование
  финальной системы; для исследовательских и учебных целей ограничений нет.
- MiewID использован исключительно в сравнительном эксперименте (A/B) и в финальную систему
  не входит; веса загружаются с `trust_remote_code=True`, commit hash зафиксирован.
- Компоненты под AGPL-3.0 (Ultralytics YOLO, FastSAM) учитываются при выборе лицензии проекта.
